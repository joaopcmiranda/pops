# Brazil Trip Content

This directory contains all the markdown files and content for organizing your Brazil trip.

## Directory Structure

- **`destinations/`** - Information about places you want to visit
- **`itinerary/`** - Daily schedules and trip timeline
- **`transport/`** - Flight bookings, train tickets, bus schedules, etc.
- **`accommodation/`** - Hotel bookings, Airbnb details, hostels
- **`activities/`** - Tours, experiences, restaurants, attractions to visit
- **`budget/`** - Cost breakdowns, expenses tracking
- **`documents/`** - Important travel documents, visas, insurance info

## How to Use

Each directory can contain markdown files that will be automatically loaded by the trip organizer app. You can create files like:

- `destinations/rio-de-janeiro.md`
- `itinerary/day-1.md`
- `accommodation/copacabana-hotel.md`
- `activities/christ-the-redeemer.md`

The app will parse these files and display them in an organized, searchable interface.
