# POps Event Manager

## Overview
Event management application within the POps suite, leveraging React 19, TypeScript, Vite 7, and shadcn/ui components.

## Key Features
- Event creation and management
- Calendar view
- Venue management
- Guest list tracking
- Budget tracking
- Event templates

## Development
- Port: 4004
- Framework: React 19 + TypeScript
- Styling: Tailwind CSS v4
- UI Components: shadcn/ui

### Quick Start
```bash
pnpm dev:events  # Start development server
```
